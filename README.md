# ARIMA Mortality Forecasting

## Overview
This project applies ARIMA time-series models to forecast mortality trends from 1999–2035 across multiple demographic datasets.

## Project Structure
- **data/** – input datasets used for modeling
- **notebooks/** – ARIMA analysis and forecasting workflow
- **results/** – model outputs and evaluation metrics
- **plots/** – forecast visualizations and trend projections

## Methods
- Time-series preprocessing
- Stationarity testing
- ARIMA parameter selection
- Forecast generation with confidence intervals

## Tools
Python, Pandas, Statsmodels, Matplotlib

## Objective
To analyze long-term mortality patterns and generate interpretable forecasts using statistical time-series modeling.
