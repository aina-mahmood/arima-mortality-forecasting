ARIMA model output results for mortality forecasting analysis.

This folder contains processed model outputs generated after fitting ARIMA time-series models on multiple demographic datasets.

Contents include:
- Model summaries and selected ARIMA parameters
- Forecast values and confidence intervals
- Evaluation metrics used for model validation

Datasets covered:
- Census-based mortality forecasts
- Gender-specific mortality trends
- Race-based mortality forecasts
- Urbanization trend forecasting

These files represent the final analytical outputs derived from the forecasting pipeline.
